Política de Privacidad

1. Responsable

El responsable de este sitio web es el titular del mismo.

2. Datos personales

Actualmente este sitio web no solicita datos personales a los usuarios ni dispone de formularios de contacto o registro.

3. Finalidad

La finalidad de esta web es ofrecer herramientas y calculadoras gratuitas de uso informativo.

4. Conservación de datos

Al no recogerse datos personales, no se almacenan datos de los usuarios.

5. Derechos

Cuando proceda, los usuarios podrán ejercer sus derechos de acceso, rectificación, supresión y demás derechos reconocidos por la normativa vigente.

6. Cambios

Esta política podrá actualizarse para adaptarse a cambios legales o nuevas funcionalidades del sitio web.
